# Getting Started

## Download Drifty

Go to the official website of Drifty and click on the download button. It will redirect you to the download page.
![Download Design](https://github.com/user-attachments/assets/b42c540b-eb99-4b9c-9c2f-6143d40166f0)
<Tabs items={['GUI', 'CLI']}>
    <Tabs.Tab>
    - Click on **GUI** from the _Application Type_ menu.
      ![Application Type](https://github.com/user-attachments/assets/d729f074-0f73-4d79-b562-7717b673d6c7)

    - If you are using Windows, click on the download button with the **Windows** icon, and it will start downloading the executable file (**`.exe`**). You may also download the **msi** (Microsoft Installer) file by clicking on **Prefer the msi?**.
      ![Windows Download button highlighted](https://github.com/user-attachments/assets/90865f7c-b0df-4957-9560-9e8a6a8c5bfb)

    - If you are using Linux, click on the download button with the **Linux** icon, and it will start downloading the executable binary.
      ![Linux Download button highlighted](https://github.com/user-attachments/assets/aad3ad93-3126-4d77-9305-61d68a69a313)

    - If you are using macOS, click on the download button with the **Apple** icon, and it will start downloading the package file (**`.pkg`**).
      ![MAC Download Exp](https://github.com/user-attachments/assets/35cdd3b7-6ea5-43ba-895b-acf7dc972321)
    </Tabs.Tab>

## Installation

<Tabs items={['GUI', 'CLI']}>

    <Tabs.Tab>
    ### Drifty GUI

        <Tabs items={['Windows', 'Linux', 'macOS']}>
            <Tabs.Tab>
            ##### Windows MSI

            <Steps>
            ### Run the downloaded MSI file
            Double-click on the downloaded MSI file to start the installation process.

            ### Installation Wizard
            1. Click on the **Next** button to start the installation.
               ![Click on Next](https://github.com/user-attachments/assets/c7e63989-77cb-44a5-a86e-7dec25f6f524)
            2. Choose the **installation directory** and click on the **Next** button.
               ![Choose installation directory](https://github.com/user-attachments/assets/aa18ad7b-1a1e-4acb-8b88-a354f931dc78)
            3. Click on the **Install** button to start the installation.
                ![Click on Install](https://github.com/user-attachments/assets/a8ca432d-d628-480d-9ea2-a58a31936efc)
            4. Click on the **Finish** button to complete the installation.
                ![Click on Finish](https://github.com/user-attachments/assets/abdfca50-168c-4539-8ff4-aeace2dc3fe2)

            ### Start Drifty GUI
            You can start using Drifty GUI by searching for it in the Windows Start menu.
            </Steps>

            ##### Windows EXE

            For Windows exe, you can directly run Drifty GUI by double-clicking on the downloaded executable file.
            </Tabs.Tab>

            <Tabs.Tab>
            ##### Linux

            Open the terminal and navigate to the directory where the binary is downloaded and run the following command to start Drifty GUI:

            ```bash
            chmod +x Drifty-GUI_linux
            ./Drifty-GUI_linux
            ```
            </Tabs.Tab>

            <Tabs.Tab>
            ##### macOS

            <Steps>
            ### Run the downloaded package file
            Double-click on the downloaded package file to start the installation process.

            ### Installation Wizard
            1. Click on the **Continue** button to start the installation.
               ![Click on Continue](https://github.com/user-attachments/assets/4f2c22d7-549c-4730-accf-d8329ab60e21)
            2. Select the **installation directory** and click on the **Continue** button. You can also change the installation directory by clicking on the **Change Install Location** button.
               ![Select installation directory](https://github.com/user-attachments/assets/495a977a-ab66-40e2-bb25-73b171d38877)
            3. Click on the **Install** button to start the installation. You may need to enter your system password to proceed with the installation.
               ![Click on Install](https://github.com/user-attachments/assets/015946f0-7c62-4ebc-bed9-bdcbf2efdd13)
            4. Click on the **Close** button to complete the installation.
               ![Click on Close](https://github.com/user-attachments/assets/931f74b4-4753-4f71-ba5f-6a584b302ad8)

            ### Start Drifty GUI
            You can start using Drifty GUI by searching for it in the applications' menu.
            </Steps>
            </Tabs.Tab>
        </Tabs>
    </Tabs.Tab>

    <Tabs.Tab>
    ### Drifty CLI

        <Tabs items={['Windows', 'Linux', 'MacOS']}>
            <Tabs.Tab>
            ##### Windows

            Open the terminal and navigate to the directory where the binary is downloaded and run the following command to start Drifty CLI:

            ```bash
            Drifty-CLI.exe
            ```
            </Tabs.Tab>

            <Tabs.Tab>
            ##### Linux

            Open the terminal and navigate to the directory where the binary is downloaded and run the following command to start Drifty CLI:

            ```bash
            chmod +x Drifty-CLI_linux
            ./Drifty-CLI_linux
            ```
            </Tabs.Tab>

            <Tabs.Tab>
            ##### MacOS

            Open the terminal and navigate to the directory where the binary is downloaded and run the following command to start Drifty CLI:

            ```bash
            chmod +x Drifty-CLI_macos
            ./Drifty-CLI_macos
            ```
            </Tabs.Tab>
        </Tabs>
    </Tabs.Tab>
</Tabs>
