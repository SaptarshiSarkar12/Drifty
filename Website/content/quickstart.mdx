# Quickstart

This quickstart guide will help you install and run Drifty on your system in just a few minutes. We'll cover:

1. Downloading the right version for your system
2. Installing Drifty

## Download Drifty

Select the version that matches your operating system and preferred interface (GUI or CLI) from the download options below:

<DownloadsIframe src="/download#download-assets" />

   ![Windows MSI installation - 4](https://github.com/user-attachments/assets/fe5972d5-3f75-4a58-a04b-a2fbe7b6e5ff)
